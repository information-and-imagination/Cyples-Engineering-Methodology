Originals:

This really all stemed from a ms access db/app (pre 2001?) i was working on that I was trying to save models for povray. I just selected the povray object defined it's dimensions and it would generate the povray code. then I was in middle of getting it to generate the code to do the same object with the same dimensions in DesignCAD 3d which I have and had a license for since 92 or 93. As I came across other object systems, direct X, autocad plan was to add those options as well. Now that AI is here that is no longer needed but to set up this type of Concept diagram it is a great way to give AI some rules and constraints to help keep it focused.
